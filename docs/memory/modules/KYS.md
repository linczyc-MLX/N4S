# Module: KYS (Know Your Site)

> **Status**: ✅ Live | **Header**: #374151 | **Last Updated**: 2026-02-22

## Purpose
Site assessment and compatibility scoring. Evaluates potential building sites against client requirements and program needs. Site-Vision Compatibility Assessment framework.

## Key Files
- `src/components/KYS/KYSModule.jsx` — Module root
- `src/components/KYS/KYSModule.css` — Module styles
- `docs/KYS-SPECIFICATION.md` — Full module spec

## Key Patterns
- Multi-criteria site scoring (topography, access, views, utilities, regulatory)
- Integration with FYI space program for site capacity analysis
- Site typology classification (coastal, alpine, urban, suburban, rural)
